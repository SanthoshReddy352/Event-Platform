# EventX — User Guide (Copy‑Paste Ready)

## Quick Start (for participants)
1. Visit Home → Browse Events
2. Click event → Read details and registration form
3. Fill the registration form and submit
4. If payment required, complete payment (Razorpay)
5. Receive confirmation email and ticket (if issued)

## Account management
- Sign up / Login using email.
- Edit profile from My Account → Profile.
- View registrations at My Account → My Events.

## Registration flow
```
1. Open event page
2. Click 'Register'
3. Complete required fields and upload files if requested
4. Pay (if required) → you'll see payment confirmation
5. Wait for approval (if event requires manual approval)
6. Receive email notification on approval/rejection
```

## Cancelling registration
- Go to My Events → Select registration → Cancel
- Refunds handled by Admin via Payments Dashboard (subject to policy).

## Help & Support
- Contact support@eventx.example for help.
- For urgent issues, use the Admin contact listed on the event page.
